# Deluxe Chatbot (React + Vite)

A clean, dark, minimal ChatGPT‑style chat UI. Type messages and the bot replies “Thanks for chatting!” every time.

## Quick start

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

## Features

- Modern, modest dark theme
- Sticky header, chat viewport, and input panel
- Enter to send (Shift+Enter for newline)
- Auto‑scroll to latest message
- Lightweight: React + Vite, zero extra UI libs
